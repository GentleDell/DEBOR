# DEBOR
Implementation of dressed expressive body reconstruction (DEBOR)

# Dependencies
## Required
- python 3.7
- pytorch
- tensorboardx
- numpy
- scipy
- opencv
- matplotlib
- tqdm
- yaml
- chumpy
- trimesh
- addict  (required by open3d)
- plyfile (required by open3d)
- sklearn (required by open3d.ml3d)
- pandas  (required by open3d.ml3d)
- open3d
